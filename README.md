# Not-a-wiki

A very simple wiki implementation using Turso as backend.

You must provide a`.env` file with these properties:

```ini
TURSO_DATABASE_URL=libsql://...
TURSO_AUTH_TOKEN=ey...
```

## Features

* Display a single page
* Edit a single page

## Missing

* Edit protection
* History
* Modified date
* Recent changes
* Image upload
